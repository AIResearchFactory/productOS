# Case 01 — Feature Discovery Brief

## Goal
Create a decision-ready discovery brief in <15 minutes.

## Demo Prompt
```
Read @context.md and create a discovery brief for "Smart Notifications".
Include:
1) user problems
2) top 3 hypotheses
3) risks
4) what to validate this week
5) recommendation: build / test / drop
```

## Expected Output Artifact
- `smart-notifications-discovery-brief.md`

## Success Criteria
- Contains explicit recommendation
- Contains this-week validation plan
- Mentions at least 3 risks
