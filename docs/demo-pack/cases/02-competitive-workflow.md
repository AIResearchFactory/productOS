# Case 02 — Competitive Benchmark Workflow

## Goal
Research 5 competitors in parallel and synthesize a comparison.

## Competitor Input
- Notion
- Coda
- Airtable
- Monday
- ClickUp

## Synthesis Prompt
```
Synthesize all competitor reports into:
1) feature matrix
2) pricing matrix
3) enterprise-readiness score (1-5)
4) strategic gaps we should exploit next quarter
```

## Expected Output Artifacts
- `competitor-notion.md` ... `competitor-clickup.md`
- `competitive-comparison.md`

## Success Criteria
- Parallel iteration is enabled
- Comparison includes all 5 competitors
