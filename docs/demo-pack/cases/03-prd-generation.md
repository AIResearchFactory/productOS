# Case 03 — PRD from Existing Artifacts

## Goal
Convert discovery + competitive insights into PRD v1.

## Demo Prompt
```
Use @smart-notifications-discovery-brief.md and @competitive-comparison.md.
Generate a PRD with:
- problem statement
- target users
- scope (in/out)
- user stories
- acceptance criteria
- success metrics
- rollout plan
```

## Expected Output Artifact
- `smart-notifications-prd-v1.md`

## Success Criteria
- Has measurable success metrics
- Has acceptance criteria
- Has explicit out-of-scope section
