# OpenCode Smoke Artifact

## Goal
Verify that OpenCode is properly installed and functioning correctly by executing a simple smoke test.

## Steps
1. Launch OpenCode in the workspace
2. Verify basic commands work (read, write, edit files)
3. Confirm tool responses are accurate
4. Test file system operations

## Expected Outcome
- All core tools respond without errors
- File operations complete successfully
- No unexpected crashes or failures
- System is ready for productive use
